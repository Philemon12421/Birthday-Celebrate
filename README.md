# 🌟 Radiant Celebration: An Interactive Birthday Odyssey

A premium, highly interactive full-stack celebration card designed with meticulous typographic pairings, modern web animation techniques, ambient physical synthesizers, and modular component structures. 

This application takes the recipient through a multi-sensory journey, combining delicate micro-interactions with playful challenges to offer land-locked birthday wishes.

---

## ✨ Features & Interactive Chapters

### 1. **Ambient Auditory Engine (Synthesized & Orchestrated)**
*   **Generative Sound Design**: Includes an on-demand volume synthesizer that drives ambient background themes.
*   **Haptic Auditory Cues**: Integrated pop, glitter transitions, and physical sound-fx that play instantly during action sequences (such as candle-blowouts and scratching milestones).

### 2. **Typewriter Wishes & Hand-Written Treasures**
*   **Evolving Typewriter Windows**: Elegant typewriter text simulations rendered at optimal, fast-typing modern speeds for an engaging letter-unfolding effect.
*   **Handwritten Envelopes & Card Slips**: Styled with the delicate **"Poppins"** signature font families, layered under gorgeous fading transitions.

### 3. **The Polaroid Memories Gallery (Drag & Drop + Upload)**
*   **Interactive Slideshow Deck**: A fully interactive Polaroid slider utilizing high-contrast vector backings.
*   **Custom File-Reader Interface**: Supports dragging and dropping real-life JPEG/PNG images directly onto the card frames, or manual file uploads.
*   **Custom Celebration Backdrops**: Dynamic text pairings tailored dynamically to complement each photo card's aesthetic concept.

### 4. **The Golden Stardust Scratch Card (Chapter 6)**
*   **Metalli-Gold Foil Shader**: Programmed using high-fidelity HTML5 Canvas gradients, complete with ambient golden glitter flakes.
*   **Stardust Spray Brush**: Customized scratching brush that leaves a trail of glowing, drifting, and rotating golden vector stardust particles.
*   **Percentage-Based Reveal**: Dynamically evaluates the pixels cleared. Once 55% of the foil is cleared, it unlocks the cake.

### 5. **Blowing Out the Candles & Manifesting Wishes**
*   **Interactive SVG Cake**: Programmed three-tier vectors complete with glowing, pulsing flames with varied flicker cycles on click.
*   **Fading Wish Prompt**: Once the 3 candles are extinguished, a luxurious "Make a Wish" block slowly fades into view.
*   **Particle Canvas Confetti Shower**: Clicking the wish triggers a screen-wide, physics-bound confetti explosion, dispersing 160 rect, circle, and star-shaped metallic flakes that swing and wobble realistically under floaty gravity.

---

## 🎨 Design System & Aesthetic Choices

*   **Typography**: Clean sans-serif headings for general UI paired with high-concept serif elements for handwritten feels. 
*   **Outer Space & Focus Boundaries**: Strictly adheres to unified clean-slate displays. Cards are nestled inside centered layouts with generous, balanced margins.
*   **Micro-Animations**: Uses `motion/react` for buttery smooth scale entrances, spring-based sliding carousels, and blurring exit properties.

---

## 🛠️ Built With

*   **React 18 & TypeScript**: For scalable, robust architectural standards and complete type-safety.
*   **Vite**: Next-generation front-end bundler generating optimized production compile states.
*   **Tailwind CSS**: High-utility styling allowing responsive adjustments, fluid layouts, and distinct shadow overlays.
*   **Motion**: Premium transitions and micro-state triggers.
*   **Lucide React**: Clean, modern vector icon library.

## 🚀 Running Locally

1. Install modern dependencies:
   ```bash
   npm install
   ```

2. Run the development environment:
   ```bash
   npm run dev
   ```

3. Compile the production bundles:
   ```bash
   npm run build
   ```

4. Check type-safety standards and code linting:
   ```bash
   npm run lint
   ```
